KidFriendlyButton

Component Sheet: Oversized interactive elements for K-2 users.

Core Variants

1. Primary Lime

Used for main actions and exciting next steps.

2. Secondary Cream

Used for creative activities and secondary choices.

3. Ghost (Outlined)

Used for less prominent actions like returning.

4. Disabled State

Indicates an action is currently unavailable.

Interaction States

Focused (Keyboard / Voice)

Features a 3pt lime glow offset by a surface-colored ring for high visibility.
